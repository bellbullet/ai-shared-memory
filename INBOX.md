# INBOX

最終更新: 2026-06-27

整理前の GitHub リンク、OSS、論文、アイデア、調査候補を一時保管する場所です。

## 使い方

- まだ分類が決まっていない情報をここに置く。
- 公開できる情報だけを書く。
- 整理後は `AI_DRAWERS.md` / `PROJECTS/` / `NOTES/` へ移す。
- 移動済みの項目は削除するか、移動先リンクを残す。

## 追加フォーマット

```markdown
### Name

URL:

分類候補:

概要:

用途:

状態:

優先度候補:

メモ:
```
